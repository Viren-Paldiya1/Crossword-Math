Idea 01

What is a matrix?

A matrix is simply a rectangular grid of numbers, arranged in rows and columns. A matrix with m rows and n columns is said to have order m × n.

The grid on the left is a 2 × 3 matrix — 2 rows, 3 columns. Each individual number is called an element, located by its row and column position.

Matrices show up anywhere data needs a grid — spreadsheets, image pixels, systems of equations, and transformations in computer graphics.

In simple words

A matrix is really just a labelled table of numbers — order tells you its shape, exactly the way "rows × columns" describes any grid.

Idea 02

Addition and scalar multiplication

Two matrices can only be added if they share the exact same order — then you simply add each pair of matching elements:

[1 2; 3 4] + [5 6; 7 8] = [6 8; 10 12]

Scalar multiplication is even simpler — multiply every single element by the same number: 3 × [1 2; 3 4] = [3 6; 9 12].

In simple words

Addition only works "position to position," and scaling just stretches every entry equally — no element gets special treatment.

Idea 03

Matrix multiplication — row meets column

Matrix multiplication doesn't multiply matching positions — instead, each element of the result comes from a row of the first matrix meeting a column of the second, multiplying pairwise and adding.

Example: top-left of the answer = (1×5) + (2×7) = 19. Top-right = (1×6) + (2×8) = 22, and so on.

Two matrices can only be multiplied if the first's column count matches the second's row count — and unlike ordinary numbers, AB ≠ BA in general.

In simple words

Walk along a row, walk down a column, multiply as you go and add up the products — that single "row meets column" motion is the entire rule.

Idea 04

The determinant, and when an inverse exists

For a 2×2 matrix [a b; c d], the determinant is a single number that captures a lot about the matrix:

det = ad - bc

If det ≠ 0, the matrix has an inverse — a matrix that "undoes" it, the way division undoes multiplication. If det = 0, no inverse exists; the matrix is called singular.

The inverse of a 2×2 matrix is A⁻¹ = (1/det) × [d -b; -c a].

In simple words

The determinant is a quick health check — a nonzero value means the matrix is "reversible," a zero means it has permanently collapsed information you can't recover.

Idea 05

Vectors: quantities with a direction

A vector is a quantity with both magnitude (size) and direction — unlike a plain number (a scalar), which is only size. Speed is a scalar; velocity is a vector, because it also says which way.

A vector from the origin to point (4, 5) is written v = 4i + 5j, or simply (4, 5). Its magnitude (length) uses the same idea as the distance formula:

|v| = √(4² + 5²) = √41

In simple words

A vector is an arrow, not just a number — "how far" and "which way" both matter, and dropping either one loses information.

Idea 06

Adding vectors, and the dot product

To add two vectors, place the tail of the second at the head of the first — the shortcut from start to finish is the sum a + b (the triangle law). Component-wise, it's just as simple: add matching parts, (a₁+b₁, a₂+b₂).

The dot product of two vectors combines them into a single scalar number, and reveals the angle between them:

a · b = a₁b₁ + a₂b₂ = |a||b| cosθ

If a · b = 0, the two vectors are exactly perpendicular — a fact used constantly in physics and 3D graphics.

In simple words

Adding vectors is "follow one arrow, then the next." The dot product is a clever shortcut that tells you how aligned two arrows are, in one number.

Quick recap

Two topics, one habit of mind.

Matrices and vectors look different on the page, but both are about organising numbers so operations become mechanical, repeatable steps.
